# **App Name**: NutriPlanner

## Core Features:

- Recipe Creation: Create new recipes by adding ingredients, preparation instructions, and nutritional information.
- AI Recipe Suggestion: Suggest recipes based on available ingredients and dietary preferences using a generative AI tool.
- Weekly Meal Planning: Plan weekly meals by dragging and dropping recipes into a meal planner grid.
- Nutritional Analysis: Calculate the total macronutrients (calories, protein, carbs, fats) for each meal and day in the meal plan.
- Ingredient Database Integration: Search and import ingredients and nutritional information from an Excel database.
- Recipe Library: Store and manage a library of created and imported recipes for easy access and planning.
- Recipe macros updating: Dynamically update the macro calculations upon adding or removing ingredients. Also will re-calculate the total macros for a meal or day.

## Style Guidelines:

- Primary color: Forest green (#16A34A) to evoke health and nature.
- Background color: Mint green (#F0FDF4) to create a fresh, calming ambiance.
- Accent color: Soft teal (#2DD4BF) to highlight important elements.
- Body and headline font: 'Inter' sans-serif for a clean, modern reading experience.
- Use clean, line-based icons to represent different food categories and actions.
- Implement a bento-box style layout with clear sections for recipe creation, meal planning, and ingredient database.
- Use subtle transitions and animations to enhance user interactions, such as adding ingredients or dragging and dropping recipes.
