# Preview Project Workflow

This workflow details how to start the development preview server to run and interact with the application.

## Step 1: Start Dev Server
Start the development server with Hot Module Replacement (HMR) enabled:
```bash
npm run dev
```

## Step 2: Open browser
By default, the server starts on:
*   [http://localhost:3000](http://localhost:3000)

Open the URL in your browser to preview the app.
