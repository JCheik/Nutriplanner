# Build Project Workflow

This workflow describes how to install dependencies and build the NutriPlanner Next.js application.

## Step 1: Install Dependencies
To install all required npm packages, run:
```bash
npm install
```

## Step 2: Typecheck Code
To verify typescript health:
```bash
npm run typecheck
```

## Step 3: Production Build
To create a production-ready optimized build:
```bash
npm run build
```
